# TODO-APP

## quickstart

```
$ npm run install
$ npm run start
```

## calling the api

### Get the todolist

```
$ curl http://localhost:3000/api/todoList -i
```

## stack/technologies used

- typescript (javascript ES6)
- express, express-session
- mongodb
- bcrypt
