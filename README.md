# TODO-APP

## quickstart

```
$ npm run install
$ npm run start
```

## calling the api

### Get the todolist

```
$ curl http://localhost:3000/api/todoList -i
```

## stack/technologies used

- Typescript (Javascript ES6)
- ExpressJS
- MongoDB
